# Todo App 📝

A simple JavaScript To-Do app with local storage.

## Features ✨
✔️ Add tasks  
✔️ Edit tasks  
✔️ Mark tasks as complete  
✔️ Delete tasks  
✔️ Persistent storage using LocalStorage  

## Technologies Used 🛠️
- HTML
- CSS
- JavaScript
- LocalStorage
